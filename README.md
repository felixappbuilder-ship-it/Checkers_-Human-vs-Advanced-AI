# Checkers: Human vs Advanced AI

This is a Checkers game where a human player can play against an advanced AI.  
The game features a smart AI opponent and an intuitive interface for an enjoyable experience.

## Features

- Human vs AI gameplay
- Advanced AI algorithm for challenging moves
- Clean and simple interface
- Works on desktop browsers

## Project Structure

- `index.html` – Main HTML file  
- `style.css` – Game styling  
- `script.js` – Game logic and AI  
- `asset.draft.png` – Game assets  
- `.gitignore` – Ignore unnecessary files
